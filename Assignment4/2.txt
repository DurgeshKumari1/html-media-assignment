b) What are HTML Entities? List out 5 commonly used HTML entities.

HTML Entities
An HTML entity is used to display invisible characters and reserved characters that would otherwise be interpreted 
as HTML code. It is a piece of text, or string, that begins with an ampersand (&) and ends with a semicolon (;).

5 Commonly Used HTML Entities are - 

1. &lt; - Represents the less-than symbol <.
Example: <h2>The less-than sign: &lt;</h2>

2. &gt; - Represents the greater-than symbol >.
Example: <h2>The greater-than sign: &gt;</h2>

3. &amp; - Represents the ampersand symbol &.
Example: <h2>The ampersand (and) sign: &amp;</h2>

4. &quot; - Represents a double quotation mark "
Example: <h2>The double quote sign: &quot;</h2>

5. &copy; - Represents the copyright symbol Â©.
Example: <h2>The copyright sign: &copy;</h2>
