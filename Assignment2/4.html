List of HTML Tags

  1. a: Defines a hyperlink
  2. b: Defines bold text
  3. br: Defines a single line break
  4. em: Defines emphasized text
  5. h1 to h6: Defines HTML headings
  6. hr: Defines a thematic change in the content
  7. mark: Defines marked/highlighted text
  8. ol: Defines an ordered list
